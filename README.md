# Pattern stability in reaction–diffusion systems depends on path entropy

This repository contains code and data used to analyze rare transitions and pattern stability in nonequilibrium reaction–diffusion systems.

Code and data supporting the preprint by  
Eric R. Heller and David T. Limmer.

**arXiv:** [2603.11192](https://arxiv.org/abs/2603.11192)

## Repository contents

- **Plots/** — Data and plotting scripts for all figures in the main text and Supporting Information (SI)

- **SampleCode/** — Python program for optimizing the instanton and computing the instanton rate constant of the 1D spatially extended Schlögl model (SI Sec. SI.3.B)

- **Movies/** — Videos of the time evolution along instanton trajectories for the 1D Schlögl, 2D Schlögl, and 2D enzyme systems
