# Reaction-diffusion systems
Code and data supporting the manuscript: "Pattern stability in reaction–diffusion systems depends on path entropy"

Folder "Plots": Data and plotting programs for all Figures in the main text and the SI.

Folder "SampleCode": Python program for optimizing the instanton and calculating the instanton rate constant of the 1D extended Schlögl model in SI section SI.3.B.

Folder "Movies": Videos of the time evolution along instantons in the 1D Schlögl, 2D Schlögl, and 2D Enzyme systems.
